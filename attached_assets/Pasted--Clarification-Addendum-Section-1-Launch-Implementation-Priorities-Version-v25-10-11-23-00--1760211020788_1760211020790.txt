⚙️ 2. Profile Completion Wizard

Trigger:

Automatically after first successful login/registration through WytPass (Google or Email OTP).

Popup or overlay wizard prompting user to complete profile basics.


Flow Steps:
1️⃣ Basic Information (name, location, occupation).
2️⃣ MyNeeds (what they are looking for / buying).
3️⃣ MyOffers (what they are selling / offering).
4️⃣ Connect Socials (optional).

WytPoints Integration:

Completing profile → +10 points.

Adding MyNeed → +5 points.

Adding MyOffer → –5 points (spending points).

Points updates reflect live in the WytWallet tab.
✅ The wizard must record partial progress so users can resume from MyPanel.
---

🧠 3. WytPoints Logic (Phase 1 Rules)

Action	Points	Type

Register / First Login	+10	Earn
Complete Profile	+10	Earn
Post a Need	+5	Earn
Post an Offer	–5	Spend
Daily Login (Streak)	+1	Earn


Points are visible in the header (badge icon) and WytWallet tab.

Transactions logged in backend (points_transactions).
✅ This builds instant engagement and a reason for users to act.

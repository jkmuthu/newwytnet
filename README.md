# WytNet - Multi-Tenant SaaS Platform Foundation

WytNet is a production-ready foundation for building multi-tenant SaaS applications with low-code capabilities. It provides CRUD builders, CMS functionality, app composition tools, and cross-tenant hub management.

## 🚀 Quick Start (One-Click Setup)

1. **Clone and Install**
   ```bash
   git clone <repository-url>
   cd wytnet
   npm install
   ```

2. **Setup Database**
   ```bash
   npm run db:push
   ```

3. **Start Development**
   ```bash
   npm run dev
   ```

4. **Access the Platform**
   - Frontend: http://localhost:5000
   - API: http://localhost:5000/api
   - Login via Replit Auth (automatic redirect)

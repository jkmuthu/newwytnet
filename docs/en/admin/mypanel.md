# MyPanel - Personal User Dashboard

**MyPanel** is the individual user's personal workspace within the WytNet platform. It provides access to personal productivity tools, apps, and settings for managing individual workflows and data.

## Overview

MyPanel serves as a centralized hub where users can:
- Access personal WytApps subscribed to individually
- Manage personal profile and preferences
- Track personal tasks, events, and activities
- Connect with other users socially
- View personal analytics and insights

**Access URL**: `https://your-hub-domain.com/mypanel`

**Authentication**: Requires WytPass login (Email/Password, Google OAuth, LinkedIn OAuth, or Email OTP)

## User Workflows

### First-Time User Setup

1. **Sign Up** via WytPass (Email, Google, or LinkedIn)
2. **Complete Profile**: Add photo, bio, preferences
3. **Browse Apps**: Explore available WytApps in MyPanel
4. **Subscribe to Apps**: Choose free apps or subscribe to premium apps
5. **Customize Dashboard**: Arrange widgets and quick actions
6. **Start Using**: Access apps and begin personal productivity journey

### Daily Usage Pattern

1. **Login** to MyPanel dashboard
2. **Check Notifications**: Review updates from apps
3. **View Today's Agenda**: Check WytCalendar for scheduled events
4. **Work on Tasks**: Use WytTask to manage daily to-dos
5. **Track Activities**: Log fitness, meals, habits via respective apps
6. **Connect Socially**: Use WytConnect and WytChat for networking
7. **Review Analytics**: Check personal productivity stats at end of day

### App Subscription Workflow

1. **Browse Apps**: Navigate to App Launcher
2. **View App Details**: Read app description, features, pricing
3. **Try Free Trial** (if available): Test app for 7-14 days
4. **Subscribe**: Choose monthly or annual plan
5. **Payment**: Complete payment via Razorpay
6. **App Activation**: App appears in MyPanel immediately
7. **Start Using**: Configure app settings and begin using

## Privacy & Data Security

MyPanel implements strict privacy controls:

**Data Isolation**:
- Personal data is isolated from organization data
- Row Level Security (RLS) ensures no data leakage
- Each user's data is accessible only to them

**GDPR Compliance**:
- Right to access personal data
- Right to data portability (export)
- Right to erasure (delete account)
- Right to rectification (edit data)

**Encryption**:
- Data encrypted in transit (HTTPS/TLS)
- Passwords hashed with bcrypt
- Session cookies httpOnly and secure
